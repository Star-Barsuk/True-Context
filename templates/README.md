# `.contextignore` templates

Use a single file: **[`.contextignore`](.contextignore)** — copy it to your repository root.

Stack-specific filenames (`.contextignore_python`, etc.) were merged into this template. Enable the section you need or add rules at the bottom.

```bash
cp templates/.contextignore /path/to/your/project/.contextignore
```

See also [`.contextignore.copy`](../.contextignore.copy) in the repo root for pattern examples and negation demos.
